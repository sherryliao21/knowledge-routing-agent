We are developing a new system to automatically identify and monitor changes using artificial intelligence (AI) on imagery. This system will improve our ability to track potential issues and enhance overall data security by upgrading our old file management system. We anticipate the project officially starting in mid-May.

The new system will provide better tools for monitoring identified points and managing follow-up actions.
